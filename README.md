# ClockHub

- Una plataforma de gestión de horarios y calendarios.

## Tecnologias

- Next.js
- TypeScript
- TailwindCSS
- Prisma
- PostgreSQL
- JWT
- Zod
- Cookie

## Instalacion

```bash
    npm install
    npm run dev
```
